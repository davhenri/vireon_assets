🚀 Willkommen beim Vireon Missions-System!

Du bist Pilot der Raumstation Vireon. Deine Aufgabe: Navigiere dein Schiff durch den Weltraum, weiche Asteroiden aus, sammle Energie-Kristalle und erreiche deine Ziele.

Hinweis: Wähle eine Mission aus dem Dropdown-Menü unten. Dein Code wird automatisch gespeichert, sodass du jederzeit weitermachen kannst.

💡 Tipps & Tricks
  • Run: Führt deinen gesamten Code aus
  • Step: Führt nur einen Schritt aus (hilfreich zum Debuggen)
  • Continue: Führt Schritte automatisch nacheinander aus
  • Reset: Setzt die Mission zurück
  • Speed: Wähle die Ausführungsgeschwindigkeit
  • Motion: Schalte Animationen an/aus

Hinweis: Dein Code wird automatisch gespeichert. Du kannst die Seite jederzeit verlassen und später weitermachen.
